## 🧾 📌 manifest.json עם מטא־דאטה (בדיוק כמו שרצית)

ב-manifest אפשר לתמוך ב:

* 🏷️ `title`
* 📝 `description`
* 🧑 `author`
* 🧷 `tags`
* 🖼️ `thumb`
* 🎞️ `series/season/episode`

וככה הקלפים יראו “תוכן אמיתי” ולא רק שם קובץ.
