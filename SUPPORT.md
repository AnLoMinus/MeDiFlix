## 🆘 תמיכה ושאלות נפוצות

**איך מוסיפים קובץ חדש?**  
גררו וידאו/אודיו/PDF לתיקיית `media/` או לשורש. לחצו על ♻️ רענון באתר — הסורק האוטומטי יזהה את הקובץ.

**האם חייבים manifest.json?**  
לא. אם אין, האתר יסרוק את המאגר דרך GitHub API. להאצת ביצועים מומלץ להשאיר manifest.json עדכני.

**האתר לא רואה קובץ חדש ב-Pages**  
- ודאו שהקובץ נמצא ב-branch הראשי.  
- חכו לסיום Deployment של GitHub Pages.  
- לחצו רענון באתר (ללא cache).

**ערוצי קשר**  
- Issues פתוחים ב-GitHub לכל תקלה או שאלה.  
- פניות אבטחה: ראו `SECURITY.md`.  
- דיוני כיוון עתידי: ראו `ROADMAP.md`.
